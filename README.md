# 三角冒險
[english version](./README.en.md)

一個從種一棵樹開始的冒險紀錄,
由公民自主發起的公共參與.

我們相信改變社會, 從撿起身邊的一片垃圾就可以開始.

---

## 冒險計畫
* [綠三角公園](./adventures/GreenTrianglePark.md)
  - 進度: 公園計畫認領中, 與里長/里幹事口頭報告.
  - 規劃方向: 冒險公園 + 食物森林
  - 位址: [高雄市楠梓區德富街底 (接翠屏國中小)](https://www.openstreetmap.org/way/884708306)
